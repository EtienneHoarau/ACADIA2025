Prérequis : 
- avoir ffmpeg installé sur son pc, tuto : https://lecrabeinfo.net/installer-ffmpeg-sur-windows.html
- pour lancer les fichiers .sh


Comment utiliser?
Mettre le fichier .sh dans un dossier comprenant : 
- un dossier vide nommée augmented_images
- un dossier avec les images à modifier pour en augmenté son nombre nommée "good"

Lancer le fichier .sh
